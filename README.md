# Usage

**RUN AS ADMINISTRATOR**

## standalone mode

### start

```
> ./caffeine
```

### termination

Ctrl+C

## app mode

### start

```
> ./caffeine run PATH_TO_EXE
```

### termination

terminate the app

## shortcut-creation mode

### preparation

```
> ./caffeine PATH_TO_EXE
```

### start

start the shortcut

### termination

terminate the app
